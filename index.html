❓ Help Topics
✕
Noise Types:
• Fiber — diagonal strand texture. Angle = direction. Density = strand count.
• Stucco — bumpy plaster. Higher roughness = chunkier bumps.
• Lava — Voronoi cellular. Dark seams between glowing cells.
• Water — layered sine ripples. Density = wave directions stacked.

Emboss / Light:
• Depth — surface relief intensity.
• Normal Scale 1 — sharpest detail. Higher = smoother wider bumps.
• Spec Hardness — high = tight pin highlight, low = wide sheen.
• Ambient — fills shadows (high = flat look).
• Cavity Shadow — AO darkening in crevices.
• Sharpen — unsharp mask post-process.

.SAI Format:
Save/load all settings as an encrypted .sai preset file. The file uses XOR encryption + base64 encoding — readable structure but opaque content. Share presets with others using TextureForge95.
